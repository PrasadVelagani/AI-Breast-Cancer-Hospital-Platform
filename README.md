# Breast Cancer Classification using Explainable AI (Grad-CAM)

This project implements a deep learning-based breast cancer classification system using histopathological images from the BreakHis dataset. The model classifies images as **Benign** or **Malignant** and provides visual explanations using **Grad-CAM**.

## Features
- CNN (VGG16 Transfer Learning)
- Binary classification (Benign vs Malignant)
- Grad-CAM visual explanations
- Tkinter GUI application
- Confidence score display

## Dataset
BreakHis – Breast Cancer Histopathological Dataset  
https://www.kaggle.com/datasets/waseemalastal/breakhis-breast-cancer-histopathological-dataset

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Tkinter
- NumPy
- Matplotlib
